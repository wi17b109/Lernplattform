Level 9: Bedingungen if/else

Eine if-Anweisung ermittelt, welche Anweisung basierend auf dem Wert eines booleschen Ausdrucks auszuführen ist. Ein boolscher Ausdruck ist eine Variable, die zwei mögliche Zustände hat: true oder false. Wenn die Bedingung im if wahr, also true ist, wird der Code in der if-Anweisung ausgeführt. Wenn die Bedingung nicht wahr, also false ist, wird der Code in der else-Anweisung ausgeführt.

if (condition == true) {

} else {

}

Wenn es mehr als zwei Möglichkeiten gibt, kann zusätzlich zu if und else noch else if verwendet werden. In else if werden einfach weitere Bedingungen wie im if angegeben. Es kann unendlich viele else ifs geben.

if (i == 1) {

} else if(i == 2) {

} else {

}
