Level 5: Vererbung

In diesem Beispiel geht es nur um Theorie. Es gibt keine Aufgabe, die zu lösen ist. Vererbung ermöglicht die Erstellung neuer Klassen, die in anderen Klassen definiertes Verhalten wiederverwenden, erweitern und ändern. Die Klasse, deren Member vererbt werden, wird Basisklasse genannt, und die Klasse, die diese Member erbt, wird abgeleitete Klasse genannt. Eine abgeleitete Klasse kann nur eine direkte Basisklasse haben.

Die Vererbung wird in der abgeleiteten Klasse folgendermaßen definiert:

class Cat : Animal {

}
